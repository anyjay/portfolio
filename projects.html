VocabQuiz Java

While taking Foundations to CompSci my Sophmore year I made a code that quized you on vocab (for my literature class' trimester vocab quiz), I shared my code with the rest of the people in my class and some others that had the same teacher through replit. By the end of the year, I was able to improve the code to shuffle questions and used arrays to greatly shorten the amount of lines. It tracks your score and makes you repeat the righ answer once you get it wrong three times. I got 100 on all the quizes I studied for with my code (I could easily modify it to have different questions, Ex: I used it for Chemistry too)
